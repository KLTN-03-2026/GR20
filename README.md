## 🚀 Hướng dẫn cài đặt

### Yêu cầu

- Node.js
- Git

### Clone project về máy

```bash
git clone https://github.com/HoanBao7904/HOMELINK-AI.git
cd HOMELINK-AI
```

### Chạy Backend

```bash
cd Backend
npm install
npm start
```

### Chạy FrontEnd

```bash
cd FrontEnd
npm install
npm run dev
```

## Quy trình làm việc với Git

### Lần đầu clone về

```bash
git clone https://github.com/HoanBao7904/HOMELINK-AI.git
cd HOMELINK-AI
git checkout ten_nhanh_cua_ban
```

### Hàng ngày trước khi code

```bash
git pull origin ten_nhanh_cua_ban
```

### Sau khi code xong

```bash
git add .
git commit -m "mô tả những gì bạn đã làm"
git push origin ten_nhanh_cua_ban
```

### ⚠️ Lưu ý quan trọng

- ✅ Chỉ push lên **nhánh của mình**
- ❌ Không push thẳng lên **main**
- Commit thường xuyên để thầy theo dõi tiến độ
